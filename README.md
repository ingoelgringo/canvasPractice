## Canvas Practice

Ett litet övningsprojekt för att experimentera med partiklar i canvas.

### Vad det gör

- Regn faller över skärmen med en liten vindeffekt.
- Droppar skapar splash-effekter när de träffar botten eller logotypen.
- Klicka var som helst på skärmen för att starta och stoppa regnet.

### Teknik

- HTML
- CSS / SCSS
- JavaScript

### Syfte

Projektet är främst gjort för att öva på animationer, canvas och enkel interaktion i webbläsaren.
